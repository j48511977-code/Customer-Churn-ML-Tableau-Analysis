# Customer Churn ML and Tableau Analysis

This project focuses on customer churn prediction and visualization using Machine Learning and Tableau.

## Project Overview

The project analyses customer churn data from a telecommunications company to identify patterns and factors affecting customer retention. Python was used for machine learning implementation and Tableau was used for dashboard creation and storytelling.

## Objectives

- Perform data preprocessing and cleaning
- Conduct exploratory data analysis (EDA)
- Build and evaluate machine learning models
- Compare model performance using evaluation metrics
- Create Tableau dashboards for visualization
- Generate business recommendations to reduce churn

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Tableau

## Machine Learning Models Used

### Logistic Regression
Used for binary classification to predict whether a customer will churn or not.

### Decision Tree
Used to identify decision rules and important factors affecting customer churn.
